## Recommended Image Rules

- Use consistent lighting.
- Keep all images warm and clean.
- Make sure screenshots are readable if they are real portfolio screenshots.
- Use descriptive alt text.
- Compress large JPGs before uploading.
- Keep filenames lowercase with hyphens.
