# GitHub Upload Checklist

## For GitHub Pages

Upload these files to your repository root:

```text
index.html
styles.css
assets/
```

Use the `index.html` and `styles.css` files from the `static` folder.

## For a Next.js Site

Copy this file into your app:

```text
next-app/app/page.jsx
```

Then copy:

```text
assets/
theme/
```

## Before Publishing

- Replace placeholder portfolio projects with real case studies.
- Add your real LinkedIn and GitHub links.
- Confirm the email address is correct.
- Add a resume PDF if you want one.
- Test the site on your phone.
- Make sure every image has alt text.
