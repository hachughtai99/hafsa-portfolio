# Theme Guide

## Visual Direction

The site should feel sharp, editorial, strategic, and confident. It should look more like a consultant's brand studio than a casual resume page.

## Mood Words

- Strategic
- Editorial
- Clear
- Confident
- Minimal
- Psychology-led
- Modern

## Color Palette

| Token | Hex | Use |
| --- | --- | --- |
| Ink | `#111111` | Main text and CTA background |
| Paper | `#F8F6F3` | Main background |
| White | `#FFFFFF` | Service section and cards |
| Muted | `#5F5F5F` | Secondary text |
| Line | `#D8D2CA` | Borders |
| Sage | `#6F7F68` | Calm strategic accent |
| Clay | `#A85F4D` | Warm human accent |
| Gold | `#C49A4A` | Highlight accent |

## Typography

Recommended:

- Headings: `Inter`, `Satoshi`, `Neue Haas Grotesk`, or `Arial`
- Body: `Inter`, `system-ui`, or `Source Sans 3`

The original ChatGPT draft uses bold modern typography. Keep that. Avoid overly decorative fonts.

## Layout

- Keep the hero full-screen or near full-screen.
- Center the hero text.
- Use very large headline type on desktop.
- Keep sections spacious.
- Use three service cards.
- Make the final CTA black with white text.

## Image Style

Use images that feel like content strategy and brand systems:

- Editorial desk scenes
- Content calendars
- Laptop with analytics dashboards
- Social media planning boards
- Notebook strategy sketches
- Clean brand moodboards

Avoid:

- Generic office stock photos
- Fake influencer poses
- Overly corporate handshake images
- Neon marketing graphics
- Random social media icons everywhere

## Design Notes

The site should not feel cluttered. The copy is strong, so the design should give it space.
